How this library works

Each skill is its own folder under skills/ with an index.html inside it. The card grid above just links to them. To add one: create skills/<name>/index.html, then duplicate a card in this file and point it there. Commit, push, and it's live.

Served over HTTPS by GitHub Pages, so embedded videos and external resources load exactly like any website — no local server, no Node, nothing to run.

Replace this with your repo: github.com/your-username/learning-library
