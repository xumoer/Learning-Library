How this library works

Each skill is its own folder under skills/ with its guide inside it. The card grid above is generated from modules.js. To add one: create your page under skills/<name>/, then add one entry to the MODULES array in modules.js (copy the template at the top of that file). Commit, push, and it's live.

Served over HTTPS by GitHub Pages, so embedded videos and external resources load exactly like any website — no local server, no Node, nothing to run.
